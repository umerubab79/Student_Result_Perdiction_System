# 🎓 Student Performance Prediction System

This Streamlit application predicts student grades using socio-economic and academic factors.  
It uses an **Artificial Neural Network (MLPRegressor)** to estimate final grades and provides insights through interactive dashboards.

---

## 📂 Dataset
- Source: [UCI Student Performance Dataset](https://archive.ics.uci.edu/ml/datasets/Student+Performance)
- Records: 395 students
- Features: Demographics, family background, study habits, lifestyle, and academic history

---

## 🧠 Model
- Algorithm: Artificial Neural Network (MLPRegressor)
- Layers: 128 → 64 → 32 → 16 (ReLU activation)
- Optimizer: Adam
- Target: Final Grade (G3)

---

## ✨ Features
- Multi-page navigation (Prediction Form, Analytics Dashboard, About Project)
- Student Information (Name, Father’s Name, Mobile Number)
- Prediction results with pass/fail status and percentage
- Interactive charts (Seaborn, Matplotlib, Plotly)
- Professional UI with custom styling
- Downloadable reports (optional future feature)
